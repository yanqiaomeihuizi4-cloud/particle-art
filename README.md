# Particle Art

ブラウザだけで動くシンプルなパーティクル描画デモです。マウスやタッチの動きに合わせて、軌跡を描く光の粒子が発生します。クリック(またはタップ)すると粒子が爆発するように広がります。

## 使い方

1. [index.html](index.html) をブラウザで開きます。
2. マウス、またはタッチで画面を動かして描画します。
3. 左上のパネルで色相・サイズ・軌跡の残りを調整できます。
4. 「画面をクリア」ボタンでキャンバスをリセットできます。

## 技術

- 依存ライブラリなし、Canvas 2D APIのみで実装されています。
- `index.html` を開くだけで動作します。
